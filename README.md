# Jasper Intro Page

A Jasper Canvas-inspired portfolio page featuring an infinite canvas interface with expandable project cards.

## Features

- Infinite canvas with draggable viewport
- Expandable project cards (click to expand, click X or press Escape to close)
- Smooth animations and transitions
- Modern, clean design inspired by Jasper's Canvas interface
- Responsive design for mobile and desktop

## Local Preview

### Option 1: Using Node.js (Recommended)

```bash
npm start
```

Then open http://localhost:3000 in your browser.

### Option 2: Using Python

```bash
# Python 3
python3 -m http.server 3000

# Python 2
python -m SimpleHTTPServer 3000
```

Then open http://localhost:3000 in your browser.

### Option 3: Using PHP

```bash
php -S localhost:3000
```

Then open http://localhost:3000 in your browser.

## Customization

Edit the project cards in `index.html` to add your own projects. Each card includes:
- Icon (emoji or custom icon)
- Title
- Description
- Expanded content with project details

## Browser Support

Works best in modern browsers (Chrome, Firefox, Safari, Edge).
